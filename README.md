# IntroToCSharp_Umandal

ITELECT2 Scenario You have been hired by Codac Logistics to develop a console-based tool for their delivery drivers. The company needs to track the daily fuel expenses and delivery performance of a single vehicle over a 5-day work week. Your task is to build a tool that gathers driver profiles, validates distance data, calculates fuel efficiency ratings, and provides a financial summary to the accounting department. BTW this is made by James Umandal from 3.2 BSIT for ITELECT2
